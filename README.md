# JUMP2DIGITAL

Checkout creado con Next.js y TypeScript para la hackaton de JUMP2DIGITAL.

## Installation

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
